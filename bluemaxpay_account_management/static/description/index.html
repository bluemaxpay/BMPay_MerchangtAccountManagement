BlueMax Pay Account Management

Overview

This module provide integration between odoo and BlueMax Pay Account. This module allows to users to create their own merchant account. and manage the merchant account

Configuration

Go to the Settings > General Settings > Credentials For BlueMax Pay. Set up all credentials.

Features

  • Integrate Odoo with BlueMax Pay.
  • Create BlueMax Pay Merchant Account.
  • Manage BlueMax Pay Merchant Account
